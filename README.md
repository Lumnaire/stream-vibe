# Lumnaire Team Collaboration Practice – Movie Streaming Site

> **A hands-on frontend project** by the **Luminaire Team** to practice **collaboration, version control, Jira workflows, and clean code** using **HTML, CSS, JavaScript**, and a **movie API**.

---

## Project Overview

We’re building a **responsive movie streaming platform** inspired by modern OTT interfaces (Netflix, Disney+, etc.). This is a **practice project** to:

- Simulate real-world team collaboration
- Use **Jira** for task tracking and Agile workflows
- Follow **best practices** in folder structure, Git, and code organization
- Implement **API integration** with TMDB (The Movie Database)
